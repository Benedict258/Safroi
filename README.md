# ClauseLens - Terms of Service & Contract AI Analyzer

ClauseLens is a high-performance AI tool designed to help users understand what they are agreeing to. By leveraging Groq LPU™ Acceleration, it scans Terms of Service (ToS), Privacy Policies, and complex legal contracts to surface hidden risks, unfair clauses, and critical privacy details.

![ClauseLens Hero](https://images.unsplash.com/photo-1589829545856-d10d557cf95f?auto=format&fit=crop&q=80&w=1200)

## 🚀 Features

- **🌐 Website Analyzer**: Paste any URL to scan Terms and Privacy Policies instantly.
- **📄 Document Upload**: Drag and drop PDF, DOCX, or TXT files for deep contract audit.
- **⚡ Groq Powered**: Uses Llama-3 models for blazing fast legal reasoning.
- **🧩 Chrome Extension**: Analyze policies directly from your browser toolbar.
- **📜 History Tracking**: Your previous scans are cached locally for quick reference.
- **🌍 Translation**: One-click summary translation for international documents.

## 🎨 Visual Identity

ClauseLens features a modern, dark-themed interface built for clarity and speed:
- **Primary Palette**: Deep Space Black (`#050B10`) & Electric Mint (`#E0FEF6`).
- **Typography**: Bold, high-contrast inter-stack for maximum readability.
- **Clean UI**: No distracting glows or unnecessary shadows—just pure actionable data.

## 🛠 Tech Stack

- **Frontend**: React 18, Vite, Tailwind CSS, Framer Motion.
- **Backend**: Express (Node.js) proxy for Secure AI access.
- **Intelligence**: Groq AI API (llama-3.3-70b-versatile).
- **Extension**: Manifest V3 Chrome Extension.

## 📦 Installation & Setup

1. **Clone the project**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Configure Environment**:
   Create a `.env` file and add your `GROQ_API_KEY`.
4. **Run Development Server**:
   ```bash
   npm run dev
   ```
