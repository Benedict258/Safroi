# How to Install Safroi Chrome Extension

The Safroi extension allows you to analyze any website's legal policies instantly without leaving the page.

## Installation Steps

1.  **Download the Extension ZIP**: Click the **"Install Extension"** button in the top right corner of the Safroi web app.
2.  **Extract the ZIP**: Unzip the `safroi_extension.zip` file to a folder on your computer.
3.  **Open Chrome Extensions**: Open Chrome and navigate to `chrome://extensions/`.
4.  **Enable Developer Mode**: Toggle the **"Developer mode"** switch at the top right of the page.
5.  **Load Unpacked**: Click the **"Load unpacked"** button.
6.  **Select Folder**: Navigate to and select the folder where you extracted the extension files.
7.  **Pin Safroi**: Click the puzzle icon in Chrome and pin **Safroi** to your toolbar for easy access.

## Features
- **One-Click Analysis**: Detects the current URL and prepares it for analysis.
- **Seamless Integration**: Connects directly to the Safroi Intelligence engine.
- **Privacy First**: Only reviews public legal text when you request it.
